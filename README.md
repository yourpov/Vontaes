# vontaes

custom discord bot made for __@psychopath.w.a.glocc__ (`1393268483080458300`)


## commands

- `/dh` - calls someone to the gym with roblox profile
- `/roblox` - shares your roblox profile link
- `/role @user` - gives roles to people (dropdown menu)
- `/ban @user [reason]` - bans someone
- `/kick @user [reason]` - kicks someone
- `/timeout @user <minutes> [reason]` - timeouts someone
- `/invite` - creates server invites
- `/credits` - my credits


## permissions

the bot needs these to work:
- send messages
- manage roles
- kick/ban/timeout members
- create invites
